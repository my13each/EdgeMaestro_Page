# EdgeMaestro_Page

Public site for **Edge Maestro** — privacy policy and product information.

Hosted via GitHub Pages at: https://my13each.github.io/EdgeMaestro_Page/

## Purpose

This repository exists solely to host public-facing pages required by app stores
(Google Play Console, etc.) and to provide a stable URL for the privacy policy.

The Edge Maestro app source code is maintained in a separate, private repository.

## Editing

The privacy policy is in [`index.md`](./index.md). Push to `main` to publish — GitHub Pages will rebuild within a minute.

## Theme

Uses Jekyll's [Cayman](https://github.com/pages-themes/cayman) theme via `_config.yml`.
